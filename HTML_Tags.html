HTML Tags

The Article Tag

The <article> tag is used to define a self-contained piece of content, which can be distributed or reused independently of the rest of the document. It can be used for blog posts, news articles, user comments, etc. The tag is typically used in combination with a heading, such as <h1>, to indicate the title of the article.

Example:

    <article>
      <h1>How to Make the Perfect Cup of Coffee</h1>
      <p>Content goes here...</p>
    </article>

The Details Tag

The <details> tag is used to create a collapsible section of content that can be expanded or collapsed by the user. It is typically used to hide additional information until the user chooses to reveal it. The tag should contain a <summary> tag that provides a brief description of the content that is hidden.

Example:

    <details>
      <summary>Click to reveal more information</summary>
      <p>Additional content goes here...</p>
    </details>

The Summary Tag

The <summary> tag is used to provide a brief summary or description of the content contained within a <details> tag. It should be used in combination with the <details> tag to create a collapsible section of content.

Example:

    <details>
      <summary>Click to reveal more information</summary>
      <p>Additional content goes here...</p>
    </details>

The Time Tag

The <time> tag is used to represent a specific time or date within a document. It can be used to markup dates, times, and durations in a machine-readable format. The datetime attribute is used to specify the date and time represented by the <time> tag in the ISO 8601 format.

Example:

    <p>The concert starts at <time datetime="2023-04-15T20:00">8:00pm on April 15th, 2023</time>.</p>
